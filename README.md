# spoorthimahadesh_2511762-_part3_regression_insights

# Task 1: Understanding the Dataset
# 1. Dependent Variable (Target Variable)

The primary objective is to identify factors influencing store sales. Therefore:

Dependent Variable: monthly_sales

This variable represents the total sales generated by each store in a given month.

# 2. Potential Independent Variables

The following variables may influence monthly sales:

marketing_spend
footfall
avg_discount_pct
staff_count
inventory_availability_pct
competitor_distance_km
holiday_flag
customer_rating
region
store_type
# 3. Numerical Variables

The dataset contains the following numerical variables:

Variable	Description
marketing_spend	Marketing expenditure
footfall	Number of customers visiting the store
avg_discount_pct	Average discount offered (%)
staff_count	Number of employees in the store
inventory_availability_pct	Percentage of inventory available
competitor_distance_km	Distance to nearest competitor
customer_rating	Average customer satisfaction rating
monthly_sales	Total monthly sales
monthly_profit	Total monthly profit
4. Categorical Variables
Variable	Description
store_id	Unique identifier for each store
month	Month of observation
region	Geographic region of the store
store_type	Type of store
holiday_flag	Indicates whether the month contains holidays (0/1)
5. Variables That May Need Cleaning or Transformation
Missing Value Treatment
competitor_distance_km may contain missing values.
customer_rating may contain missing values.
Encoding Categorical Variables

region

month should be converted into useful features such as:
Month number

For certain regression techniques, numerical features may require scaling.

# 6. Variables That May Not Be Useful for Regression
Variable	Reason
store_id	Serves only as a unique identifier and does not directly influence sales.
month	Raw date values cannot be directly used in regression without feature engineering.
monthly_profit	Highly correlated with sales and may cause data leakage if the goal is to predict sales.
